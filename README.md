01. Prime Number Checker

Description:
This program checks whether a number is prime using square root optimization.

Notes:

Written while learning Python basics.

Efficiently checks for prime numbers by iterating only up to the square root of the number.

02. Factorial Calculator

Description:

Notes:

Written while learning Python basics.
